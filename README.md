# E-Library CI4 VueJS

Sistem Informasi Perpustakaan Digital berbasis:

- CodeIgniter 4 REST API
- Vue.js 3 SPA
- Tailwind CSS
- Axios
- MySQL

## Fitur

- Login Administrator
- Dashboard
- CRUD Buku
- Upload Cover Buku
- CRUD Kategori
- CRUD User
- REST API
- Dark Mode
- Responsive Design

## Instalasi

composer install

Copy .env dan sesuaikan database

php spark serve
